# system_audio_recorder



The Flutter plugin based on flutter package [flutter_screen_recording](https://pub.dev/packages/flutter_screen_recording), and GitHub library [SystemAudioCaptureAndroid](https://github.com/HarshSinghRajawat/SystemAudioCaptureAndroid). This plugin implements the function of recording the sound played by the system on Android phones. That is, as long as an Android app is not [set to prevent other apps from recording sounds](https://developer.android.google.cn/media/platform/av-capture?hl=en#constraining_capture_by_other_apps), the plugin can record the sounds played by that app.
